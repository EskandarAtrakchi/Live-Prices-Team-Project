# Buttons

Buttons are used to perform actions in the product.

## Default

{% embed url="https://5ccbc373887ca40020446347-geedzbiswp.chromatic.com/iframe.html?id=button--basic&args=size:medium;containsIcon:false&viewMode=story" %}

```javascript
<Button
    label="Label"
    size="medium"
    kind="default"
    onClick={doTheThing}
/>
```
